# AI-ML-Engineering-Intersnship-Task
Tasks completed during AI/ML Engineering Internship covering data exploration, classification, and regression using Python and Machine Learning.

This repository contains tasks assigned during my AI/ML Engineering 
Internship. Each task focuses on a core Machine Learning concept 
implemented using Python.

Tasks Included:

Task 1 - Exploring and Visualizing a Simple Dataset
   - Loaded and inspected the Iris dataset
   - Performed exploratory data analysis
   - Visualizations: scatter plots, histograms and box plots
   - Libraries: Pandas, Matplotlib, Seaborn

Task 2 - Heart Disease Prediction
   - Binary classification on Heart Disease UCI Dataset
   - Data cleaning, EDA and model training
   - Evaluation Metrics: Accuracy, Precision, Recall, F1, Confusion Matrix
   - Libraries: Pandas, Scikit-learn, Seaborn

Task 3 - House Price Prediction
   - Regression modeling on House Price Dataset
   - Feature scaling, encoding and selection
   - Evaluation Metrics: MAE, RMSE, R2 Score
   - Libraries: Pandas, Scikit-learn, Matplotlib

Tech Stack:
   - Python 3.x
   - Pandas, NumPy
   - Scikit-learn
   - Matplotlib, Seaborn
   - Jupyter Notebook
